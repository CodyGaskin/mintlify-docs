---
title: 'GET /tags/{tagId}'
---

Retrieve the details of a specific memory tag by its ID. Requires the user's ID.

<RequestExample>

```bash Example Request
curl -X GET "https://api.memoram.app/api/v1/tags/tag_id_example...?user_id=USER_ID_FROM_CREDENTIALS" \\
  -H "Authorization: Bearer YOUR_DEVELOPER_JWT_TOKEN"
```

</RequestExample>

### Authorization

Requires `Bearer` token authentication using your Developer JWT.

### Path Parameters

<Properties>
  <Property name="tagId" type="string" required>
    The unique ID of the tag to retrieve.
  </Property>
</Properties>

### Query Parameters

<Properties>
  <Property name="user_id" type="string (uuid)" required>
    The unique ID of the user who owns the tag.
  </Property>
</Properties>

### Responses

<ResponseExample>

```json 200 OK: Success
{
  "id": "tag_id_example...", 
  "name": "work", 
  "color": "#10B981", 
  "description": "Work-related memories", 
  "is_enabled": true, 
  "created_at": "2024-01-02T11:00:00Z", 
  "updated_at": "2024-01-02T11:00:00Z"
}
```

```json 400 Bad Request
{
  "error": "Bad Request",
  "message": "Missing or invalid user_id.",
  "statusCode": 400
}
```

```json 401 Unauthorized
{
  "error": "Unauthorized",
  "message": "Invalid or expired JWT.",
  "statusCode": 401
}
```

```json 403 Forbidden
{
  "error": "Forbidden",
  "message": "Tag does not belong to the user or developer lacks access.",
  "statusCode": 403
}
```

```json 404 Not Found
{
  "error": "Not Found",
  "message": "Tag with the specified ID not found.",
  "statusCode": 404
}
```

</ResponseExample>
