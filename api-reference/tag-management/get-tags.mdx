---
title: 'GET /tags'
---

Retrieve a list of all memory tags available to the specified user. Requires the user's ID.

<RequestExample>

```bash Example Request
curl -X GET "https://api.memoram.app/api/v1/tags?user_id=USER_ID_FROM_CREDENTIALS" \\
  -H "Authorization: Bearer YOUR_DEVELOPER_JWT_TOKEN"
```

</RequestExample>

### Authorization

Requires `Bearer` token authentication using your Developer JWT.

### Query Parameters

<Properties>
  <Property name="user_id" type="string (uuid)" required>
    The unique ID of the user whose tags are being requested.
  </Property>
</Properties>
