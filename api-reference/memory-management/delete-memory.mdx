---
title: 'DELETE /memory-blobs/{blobId}'
---

Permanently delete a specific memory blob. Requires the user's ID.

This operation is irreversible.

<RequestExample>

```bash Example Request
curl -X DELETE "https://api.memoram.app/api/v1/memory-blobs/mem_blob_id_to_delete...?user_id=USER_ID_FROM_CREDENTIALS" \\
  -H "Authorization: Bearer YOUR_DEVELOPER_JWT_TOKEN"
```

</RequestExample>

### Authorization

Requires `Bearer` token authentication using your Developer JWT.

### Path Parameters

<Properties>
  <Property name="blobId" type="string" required>
    The unique ID of the memory blob to delete.
  </Property>
</Properties>

### Query Parameters

<Properties>
  <Property name="user_id" type="string (uuid)" required>
    The unique ID of the user who owns the memory blob.
  </Property>
</Properties>

### Responses

<ResponseExample>

```json 200 OK: Success
{
  "success": true
}
```

